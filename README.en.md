# QtSvgEditor

#### Description
使用Qt开发SVG编辑器。
(1)支持新建SVG文件。 
(2)支持画布设置，如大小、背景色、缩放等。
(3)支持在新建的文件中添加/编辑以下矢量元素:直线、矩形、多边形、圆、自由线、文字。
(4)支持颜色填充设置。
(5)支持编辑后保存为SVG文件。
(6)支持打开保存的SVG文件及二次编辑。
(7)支持打开通用的SVG文件。
(8)支持移动、删除矢量元素。
